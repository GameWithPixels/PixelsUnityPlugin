Detailed Description

Data Set is the set of all behaviors, conditions, rules, animations and colors stored in the Pixel die. This data gets transfered straight to the dice. For that purpose, the data is essentially 'exploded' into flat buffers. i.e. a all the keyframes of all the anims are stored in a single keyframe array, and individual tracks reference 'their' keyframes using an offset and count into that array.
