Evaluate an animation track's for a given time, in milliseconds, and fills returns arrays of led indices and colors. The returned colors are the given color modulated with the light intensity of the track for the given time. Values outside the track's range are clamped to first or last keyframe value.

Parameters
bitsThe animation bits with the keyframes data and color palette.
colorThe color for which to modulate the intensity.
timeThe time at which to evaluate the track.
retIndicesArray of LED indices to be updated.
retColorsArray of 32 bits colors to be updated.
Returns
The number of LED indices that have been set in the returned arrays.
