Detailed Description

Demonstrates various Bluetooth operations on Pixels using the device agnostic Central class. One may scan for Pixels, connect to one, retrieve information from and send messages to it.

The buttons are grayed out when the functionality it will use is not available (for example, the "write" button is not available when the application is not connected to a Pixel).
