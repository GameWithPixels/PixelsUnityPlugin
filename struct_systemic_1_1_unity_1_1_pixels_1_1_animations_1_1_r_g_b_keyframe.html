Detailed Description

Stores a single keyframe of an LED animation size: 2 bytes, split this way:

  • 9 bits: time 0 - 511 in 50th of a second (i.e )
    • 1 -> 0.02s
    • 500 -> 10s
  • 7 bits: color lookup (128 values)
